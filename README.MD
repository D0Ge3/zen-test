## `yarn`
### `yarn start`

Запуск приложения

### `yarn start {путь_к_файлу}.json`

Запуск приложения со стартовым состоянием из файла (json).  
Пример файла состояния содержится в корне проекта (example.json)

### `yarn install`

Установка необходимых зависимостей
